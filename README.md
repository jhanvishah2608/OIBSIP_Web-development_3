# OIBSIP_Web-development_3

--Objective:
To develop a simple To-Do web application that helps users add, manage, and organize daily tasks efficiently.

--Steps Performed:

1. Created the webpage structure using HTML.
2. Designed the interface with input fields, buttons, and task lists.
3. Used JavaScript to add, delete, and manage tasks dynamically.
4. Applied CSS styling to improve the appearance and user experience.

--Tools Used:

* HTML
* CSS
* JavaScript
* VS Code / Any Web Browser

--Outcome:
The To-Do web application successfully allows users to manage daily tasks through an interactive and user-friendly interface while improving front-end web development skills.
